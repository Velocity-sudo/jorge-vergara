## 🎨 Paleta de Colores

### Concepto: "Street Smart Executive"

> **Elegancia accesible con calidez latina.** Base oscura premium que evoca seriedad financiera, con acentos dorados cálidos que evocan prosperidad y logro. El verde profundo como acento secundario conecta con crecimiento y dinero. Lejos del azul corporativo frío — aquí hay alma.

### Colores Primarios

| Color | Hex | RGB | Uso |
|-------|-----|-----|-----|
| **Navy Profundo** | `#0B1D32` | 11, 29, 50 | Fondos hero, fondos principales, backgrounds oscuros |
| **Blanco Puro** | `#FFFFFF` | 255, 255, 255 | Fondos de secciones de alto contraste, tarjetas en secciones claras |
| **Blanco Cálido** | `#F8F5F0` | 248, 245, 240 | Texto sobre fondos oscuros, fondos de secciones suaves |

> **Estrategia de Contraste:** Para mantener la atención y facilitar la lectura (especialmente en móvil), intercalamos secciones de fondo oscuro (Navy Profundo) con secciones de fondo claro (Blanco Puro/Cálido).

## 🔤 Tipografía

### Fuente Principal

**Plus Jakarta Sans** — [Google Fonts](https://fonts.google.com/specimen/Plus+Jakarta+Sans)

```css
@import url('https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@300;400;500;600;700;800&display=swap');

font-family: 'Plus Jakarta Sans', sans-serif;
```

### Fuente Secundaria (Body Extended)

**DM Sans** — [Google Fonts](https://fonts.google.com/specimen/DM+Sans)

```css
@import url('https://fonts.googleapis.com/css2?family=DM+Sans:wght@300;400;500;600;700&display=swap');

font-family: 'DM Sans', sans-serif;
```

> **Rationale:** Plus Jakarta Sans es moderna, geométrica y premium. DM Sans para body text es limpia y legible. Ambas ya están en uso en el sitio actual de Jorge — las elevamos con pesos y tamaños correctos.

### Escala Tipográfica

| Peso | Valor | Uso |
|------|-------|-----|
| Light | `font-weight: 300` | Cuerpo de texto, descripciones |
| Regular | `font-weight: 400` | Body text principal |
| Medium | `font-weight: 500` | Nav links, texto destacado |
| SemiBold | `font-weight: 600` | Títulos (h2, h3), botones |
| Bold | `font-weight: 700` | Headlines principales |
| ExtraBold | `font-weight: 800` | Hero titles, números de impacto |

### Tamaños (Desktop → Mobile)

| Elemento | Desktop | Mobile (≤768px) |
|----------|---------|-----------------| 
| **Hero Title** | 4.5rem | 2.2rem |
| **Headlines (h1)** | 3.5rem | 1.8rem |
| **Section Titles (h2)** | 2rem | 1.5rem |
| **Body Text** | 1.05rem | 0.95rem |
| **Labels** | 0.7rem | 0.65rem |
| **Buttons** | 0.9rem | 0.8rem |

### Atributos Tipográficos Clave

```css
/* Headlines — tracking apretado, impactantes */
letter-spacing: -0.02em;

/* Subtitles — tracking abierto, uppercase, elegantes */
letter-spacing: 0.15em;
text-transform: uppercase;

/* Body — line-height generoso para lectura cómoda */
line-height: 1.7;

/* Números de impacto — extra bold, acento dorado */
font-weight: 800;
color: var(--brand-accent);
```

### Fuente Alternativa

`system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif` (solo si la fuente principal no carga)

## 🖼️ Dirección de Fotografía

| Atributo | Descripción |
|----------|-------------|
| **Iluminación** | Natural cálida con toques editoriales. Luz dorada/golden hour preferida. Evitar flash directo y luz fría de oficina. |
| **Mood** | Exitoso pero accesible. "Street Smart Executive" — no corporativo rígido, sino elegante casual. |
| **Composición** | Centrada para retratos, regla de tercios para lifestyle. Espacio negativo generoso. |
| **Color fotográfico** | Tonos cálidos (dorados, terracota). Evitar filtros azules fríos. Desaturar ligeramente los fondos para que el sujeto destaque. |
| **Pose/Estilo** | Business casual elevado: blazer con camiseta de calidad, sin corbata. Accesorios sutiles. Confident pero no arrogante. Sonrisa genuina, mirada directa a cámara. |

### Contextos Fotográficos Recomendados

| Contexto | Propósito | Mood |
|----------|-----------|------|
| **Retrato editorial** | Hero sections, about me | Confianza + calidez |
| **Con familia** | Contenido de humanidad (20%) | Autenticidad + fe |
| **En oficina/despacho** | Authority building | Profesionalismo |
| **Lifestyle (viajes, cena)** | Resultado de buena gestión financiera | Aspiracional |
| **Enseñando (pizarra/iPad)** | Contenido educativo | Autoridad + accesibilidad |

### Tratamiento de Imágenes Web

```css
background-size: cover;
background-position: center;

/* Overlay oscuro obligatorio sobre fotos */
.overlay {
    background: rgba(11, 29, 50, 0.55);

/* Text shadow para legibilidad */
```

## ✍️ Tono de Voz

| Atributo | Descripción |
|----------|-------------|
| **Idioma** | Español principal (80%), con Spanglish estratégico cuando conecta culturalmente |
| **Tono** | Cercano pero firme. Como un hermano mayor exitoso que quiere que ganes |
| **Perspectiva** | Primera persona ("Yo te ayudo", "Nosotros nos encargamos") |
| **Energía** | Empoderada, positiva, con urgencia cuando toca. Nunca desesperada ni "vendedora" |

### Adaptación por Segmento

| Segmento | Tono Específico |
|----------|----------------|
| **Hombres 18-30** | Directo, enfocado en crecimiento, lenguaje de negocios y hustle. "El IRS no te espera, bro." |
| **Mujeres 28-35** | Empático, colaborativo, enfocado en "nosotros". "Vamos a construir esto juntas." |
| **Familias 35-55** | Protector, tranquilizador, enfocado en legado. "Tu familia merece esa tranquilidad." |

### Frases Clave

- *"Te ayudo a hacer lo que los ricos ya saben — pero en español y sin BS."*
- *"No vendo seguros. Vendo tranquilidad financiera."*
- *"Optimizamos tus taxes para que pagues lo justo. No hay mejora, no pagas."*
- *"El sistema está diseñado para que pagues si no conoces las reglas. Yo te las enseño."*
- *"5,000+ familias confían en nosotros. Retención mayor al 90%."*

### Vocabulario de Marca

| Sí usar ✅ | No usar ❌ |
|-----------|-----------|
| "Optimizar", "Estrategia" | "Evadir", "Truco" |
| "Tu dinero", "Tu patrimonio" | "Nuestro producto", "Nuestra póliza" |
| "Tranquilidad", "Claridad" | "Miedo", "Riesgo" (como amenaza) |
| "Comunidad latina", "Nuestra gente" | "Clientes", "Usuarios" |
| "Sistema financiero", "Juego" | "Burocracia", "Complicado" |
| "Legal y organizado" | "Barato", "Descuento" |

### Reglas de Comunicación

- Oraciones cortas y directas (máximo 20 palabras)
- Voz activa SIEMPRE
- Máximo 1 exclamación por cada 500 palabras
- Párrafos de máximo 3 líneas
- Números específicos siempre: "$1,365,000 ahorrados" — NO "mucho dinero"
- Mix: 80% contenido financiero experto + 20% humanidad/fe/familia

## ❌ Usos Incorrectos

| No hacer | Por qué |
|----------|---------|
| 🚫 Usar azul corporativo frío genérico | Nos diferenciamos con navy profundo + dorado cálido |
| 🚫 Usar el dorado como color dominante | Máx 10-15%, es un acento aspiracional |
| 🚫 Tipografías no autorizadas | Solo Plus Jakarta Sans y DM Sans |
| 🚫 Fotos con filtros azules/fríos | La marca es cálida, dorada, exitosa |
| 🚫 Lenguaje corporativo/complejo | Hablamos como amigos, no como banqueros |
| 🚫 Stock photos genéricos de oficina | Usar fotos reales de Jorge, familia, comunidad |
| 🚫 Border-radius pequeño en cards | El look es redondeado (20px) |
| 🚫 Imágenes sin overlay oscuro | El texto necesita contraste |
| 🚫 Texto centrado sin max-width | Pierde legibilidad |
| 🚫 CTAs desesperados/hype | Somos premium, no vendedores de feria |
