Facebook Pixel & Eventos Meta

Guía paso a paso para instalar el Pixel y los eventos de conversión en GHL.

🧩 ¿Cómo funciona el Pixel?

Son 2 lados que deben estar conectados:

Lado ¿Dónde? ¿Qué haces?
1. El Código En GHL (en cada página) Instalas el script que ENVÍA las señales
2. Meta En Meta Events Manager Verificas que las señales LLEGAN y las usas

💡 Piensa en el Pixel como un walkie-talkie: instalas el transmisor (código) en GHL, y Meta tiene el receptor que escucha. No configuras eventos en Meta manualmente — el código los envía automáticamente.

Paso 1: Obtener tu ID de Pixel

📍 ¿Dónde encontrarlo?

  • Ve a Meta Business Suite → Events Manager
  • Tu Pixel ya debería existir (si no, creas uno nuevo)
  • Copia el Pixel ID — es un número tipo 123456789012345

Paso 2: Instalar el Pixel BASE en GHL (Header)

📍 ¿Dónde en GHL?

Abrir cada paso del embudo → ⚙️ Settings (ruedita) → "Tracking Code" → "Header"

📁 Embudo de Jorge
├── 📄 VSL (registro-1143)
└── ⚙️ Settings → Tracking → Header → [PEGAR AQUÍ]
├── 📅 Agenda (agenda-page)
└── ⚙️ Settings → Tracking → Header → [PEGAR AQUÍ]
└── 🎉 Thank You (thankyou-2381-page)
└── ⚙️ Settings → Tracking → Header → [PEGAR AQUÍ]

Paso 3: Agregar eventos EXTRA por página (Footer)

Estos van en Settings → Tracking Code → Footer de cada página específica.

📄 VSL Page → Footer

La encuesta está en esta página. 2 eventos: al tocar un campo + al enviar el form

<script>
// Evento 1: Empezó a llenar la encuesta (tocó un campo)
var formStarted = false;
document.addEventListener('focusin', function(e) {
  if (!formStarted && e.target.closest(
    'form, .ghl-form-wrap, [data-form]')) {
    formStarted = true;
    fbq('track', 'InitiateCheckout', {
      content_name: 'Encuesta VSL'
    });
  }
});

// Evento 2: Envió la encuesta completa
document.addEventListener('submit', function(e) {
  if(e.target.closest('form')) {
    fbq('track', 'Lead');
  }
});
</script>

🏷️ InitiateCheckout = empezó a llenar  |  Lead = envió la encuesta

📅 Agenda Page → Footer
