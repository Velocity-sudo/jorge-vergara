## 🎯 Principios de Diseño Aplicados

### Optimización para IA de Meta (Andromeda)
- **Keywords semánticas en cada imagen:** "impuestos", "taxes", "IRS", "errores", "multas" — el algoritmo de Meta lee el texto de los anuncios para clasificar audiencia.
- **Identificación de avatar en headline:** Cada ad identifica QUIÉN es la audiencia desde la primera línea (latinos, familias, personas que hacen taxes).
- **Señales de targeting:** El copy contiene señales que Meta usa para encontrar la audiencia correcta sin depender de intereses manuales.

### Optimización para Atención Humana
- **Contraste visual fuerte:** Navy profundo (#0B1D32) + Dorado (#D4A853) + Blanco = máximo contraste en feed.
- **Números específicos:** "$100", "$6,000", "73%", "$1,365,000+" — los números rompen el scroll instantáneamente.
- **Jerarquía tipográfica clara:** Headline grande → Sub-line mediano → CTA en botón con color de acento.
- **Social proof integrado:** Métricas de autoridad en cada imagen (5,000+ familias, $1.36M+ ahorrados, >90% retención).

### Brand Integrado
- **Paleta:** Navy Profundo + Dorado Prosperidad + Blanco Cálido (del Brand Manual)
- **Font Style:** Sans-serif bold moderno (aligned con Plus Jakarta Sans)
- **Estilo visual:** "Street Smart Executive" — premium, accesible, cálido, profesional.

## 📱 Notas por Plataforma

### Instagram Feed
- Usar formato **4:5** (ocupa más espacio en el feed = más atención).
- Si quieres usar 1:1, funciona bien para el carrusel.
- Primary Text: Máximo 125 caracteres antes del "...más" — tu primera línea es tu hook.

### Instagram Stories / Reels
- Usar formato **9:16** (pantalla completa).
- Swipe Up / Link sticker al VSL.
- Duración como story: 5-7 segundos para estáticos.

### Facebook Feed
- Funciona bien con **1:1** y **4:5**.
- Facebook muestra más texto antes de truncar — aprovéchalo con el copy extendido.

### Facebook Stories
- Usar **9:16**.
- CTA de Meta Ads funciona como botón overlay.

## 🧪 Estructura de A/B Test Recomendada

### Fase 1: Test de Ángulo (Semana 1-2)
| Campaña | Ad Set | Ads |
|---------|--------|-----|
| JV-TAXES-PROSPECTING-TEST | Broad-Latinos-FL-TX-CA | 1x Error Común (4:5) |
| | | 1x Emocional (4:5) |
| | | 1x Urgencia (4:5) |

**Objetivo:** Identificar qué ángulo tiene mejor CTR y Hook Rate. Gastar $15-20/día.

**Métricas a observar (obligatorias):**
| Métrica | Benchmark | Qué indica |
|---------|-----------|------------|
| CTR | > 1.5% | Si la gente hace click |
| Hook Rate | > 30% | Si la gente se detiene (primeros 3s) |
| CPL | < $15 | Eficiencia de costo |
| Thumb Stop Rate | > 25% | Si el creativo rompe el scroll |

### Fase 2: Test de Copy (Semana 3-4)
Tomar el ángulo ganador y probar 3 variantes de Primary Text:
1. Copy largo (como el de arriba)
2. Copy corto (3 líneas max + CTA)
3. Copy con testimonio/caso real

### Fase 3: Test de Formato (Semana 5+)
Con el ángulo y copy ganador, probar:
1. Estático (4:5) vs Reel de video
2. Estático solo vs Carrusel (3 slides)

## ⚠️ Compliance Meta Ads (Servicios Financieros)

✅ **Aprobado en estos creativos:**
- "Evaluación gratuita" — no promete ahorros específicos
- "Hacer impuestos de forma legal" — no dice "pagar menos"
- Estadísticas verificables del negocio (5,000+, $1.36M+, 90%+)
- CTAs no agresivos (ver video, evaluar, agendar)

❌ **Evitar al subir a Meta:**
- No agregar texto que prometa ahorros garantizados
- No usar palabras como "evitar impuestos" (usar "optimizar" o "hacer correctamente")
- No comparar con TurboTax/H&R Block por nombre en el ad
